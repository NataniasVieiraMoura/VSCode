A tag é usada para criar um link. O atributo href="#topo" define o destino do link como o elemento com o id="topo", que neste caso é o cabeçalho com o título "Topo da Página". Quando o link é clicado, o navegador rolará a página até o elemento com o ID especificado. A tag
