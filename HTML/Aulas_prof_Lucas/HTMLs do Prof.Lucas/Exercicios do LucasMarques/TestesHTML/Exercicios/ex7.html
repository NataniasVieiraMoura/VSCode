Enviar um Email O atributo href da tag é usado para definir o link. O valor mailto:exemplo@email.com no atributo href indica que o link abrirá o programa de email padrão do usuário com um novo email direcionado para o endereço de email especificado A tag é usada para inserir imagens na página. O atributo src especifica o caminho ou o nome do arquivo da imagem. O atributo alt fornece uma descrição alternativa para a imagem. Isso é importante para acessibilidade e também é exibido se a imagem não puder ser carregada. Certifique-se de substituir "imagem.png" pelo nome real da imagem.
