Para utilizar o DOM você:
0ª Baixe a extensão como Five sever ou preview on web server;
1ª Para utilizar click com o botão direito do mouse e click na opção 'open whith' e escolha o tipo de extenção que vai funcionar o dom.

O dom serve para visualizar o codigo funcionando emquanto você trabalha nele, assim não é necessário atualizar a página a todo momento.

Para usar você tambem pode clicar no botão 'go to live' para abrir o five server. E ao terminar de usar clicar na mesma opcão para fechar.
