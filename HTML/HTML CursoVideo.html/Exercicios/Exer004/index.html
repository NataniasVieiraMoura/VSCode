Use sites como : Unsplash, pexels,wikemidia, freepiake,hourpixel o pixabay, livreshot, wikemidiacommons...

O formato de imagems pode ser :
formato .png(imagems com transparência), formato .gif(whatsapp[premite transparência e animação.]) , formato .jpg(altacompactação para imagems não grandes[pode preder a resolução se a imagem for muito grande. Permite transparência. ]) e formato .jpeg(imagems gigantes)
